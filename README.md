# [感染圍堵模擬模型]

## Proposal Report

### 動機與目標
<!-- 說明為什麼想做這個專題 -->
此模型源自現實中的病毒擴散與圍堵，可以應用在現實中如防治傳染性疾病、阻止電力或水利等基礎設施的故障擴散影響，也能應用於資訊領域上如防堵病毒、抑制假訊息等問題，應用範圍廣泛。

專案的目標為做出一個簡易的模型，能夠對病毒的感染擴散以及防堵進行模擬，並且加入一些現實中情境的背景要素，強化模型的模擬功能。
