# 感染圍堵模擬模型

## Proposal Report

### 動機與目標
<!-- 說明為什麼想做這個專題 -->

此模型源自現實中的病毒擴散與圍堵，可以應用在現實中如防治傳染性疾病、阻止電力或水利等基礎設施的故障擴散影響，也能應用於資訊領域上如防堵病毒、抑制假訊息等問題，應用範圍廣泛。

專案的目標為做出一個簡易的模型，能夠對病毒的感染擴散以及防堵進行模擬，並且加入一些現實中情境的背景要素，強化模型的模擬功能。

### 預期功能
<!-- 列出預計實作的功能 -->

1&emsp;模擬在二維平面、給定圖上的病毒擴散。

2&emsp;模擬牆的放置(阻斷路徑)，讓病毒無法透過此條邊。

3&emsp;嘗試在有限資源下(時間、牆的數量等)最小化病毒擴散範圍。

### 使用技術
<!-- 使用的語言、框架、工具等 -->

預計使用C++撰寫。在基礎的原型做出來之後，可以嘗試利用OpenCV、SFML等框架視覺化圖形。

### 與課程的關聯
<!-- 你的專題可能涉及哪些資料結構或演算法概念？為什麼？ -->

在做病毒擴散時，會需要用到佇列以及BFS，因為病毒的擴散是由根源慢慢向外擴張。

由於最佳化此問題一般是NP-Hard，我會仿照上學期在程式設計課的經驗，用多種不同的演算法測試，比較演算法之間的差異以及不同演算法在不同輸入情況下的行為。
